# 钉钉部署指南 (H5 微应用模式)

将公文助理接入钉钉非常简单，本质上是将我们已经部署好的 Vercel 网站作为一个 H5 应用配置到钉钉工作台。

## 准备工作
1. 确保您的 Vercel 部署链接 (例如 `https://ruby-eight-80.vercel.app`) 可以正常访问。
2. 您需要拥有钉钉组织的管理员权限或开发者权限。

## 步骤一：创建应用
1. 登录 [钉钉开发者后台](https://open-dev.dingtalk.com/)。
2. 在顶部菜单选择 **"应用开发"** -> **"企业内部开发"**。
3. 点击 **"创建应用"** 按钮。
4. 填写基本信息：
   - **应用名称**：公文智能起草助手
   - **应用描述**：AI智能起草规范公文，符合GB/T 9704-2020标准
   - **应用图标**：上传一个 64x64 的图标（可截图网页 Logo 使用）
5. 确定创建。

## 步骤二：配置应用地址
1. 在应用详情页左侧菜单，点击 **"网页应用"**（或者叫 "开发管理" -> "修改"）。
2. 找到 **"应用首页地址"** 和 **"PC端首页地址"**。
3. **分别填入**您的 Vercel 部署链接：
   ```
   https://ruby-eight-80.vercel.app
   ```
   *(注意：如果钉钉要求验证域名，您可以先忽略，或者下载验证文件放到 public 目录重新部署，通常企业内部应用不需要严格的域名归属验证)*

4. 点击 **"保存"**。

## 步骤三：发布应用
1. 在左侧菜单点击 **"版本管理与发布"**。
2. 点击 **"创建版本"**。
3. 填写版本号（如 `1.0.0`）和版本描述（如 `初始版本上线`）。
4. 选择 **"应用可见范围"**（例如选择 "全员" 或特定部门）。
5. 点击 **"发布"**。

## 步骤四：在钉钉中使用
1. 打开钉钉手机版或 PC 版。
2. 进入 **"工作台"**。
3. 您应该能看到 **"公文智能起草助手"** 的图标。
4. 点击即可直接使用，体验与浏览器中一致。

## 常见问题
- **页面显示不全？** 
  应用已适配移动端，如果在钉钉内显示异常，请确保网络能访问 Vercel。
- **需要免登（自动获取名字）吗？**
  当前版本为纯 H5 模式，无需后端鉴权即可使用核心功能。如需自动填入发文人姓名，需要进行更复杂的钉钉 API 对接（涉及后端开发）。
